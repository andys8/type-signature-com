## Development

The project is implemented in [PureScript](https://www.purescript.org).

### Setup

```shell
npm install
```

### Development

```shell
npm start
```

Or if editor is using `purescript-language-server` it's enough to run:

```shell
npm run start:parcel
```

### Build

```shell
npm run build
```

### Test

```shell
npm test
```
